# RoundSquare

> The official [RoundSquare](https://roundsquare.site) homepage.

## Introduction

The homepage at [RoundSquare](https://roundsquare.site), built with [Express](https://expressjs.com/).

## Usage

Run the [Express](https://expressjs.com/) server.

```
npm start
```

Lint the source.

```
npm run lint
```
